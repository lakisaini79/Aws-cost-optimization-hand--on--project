# AWS DevOps Practice Project

This repository contains my hands-on practice of AWS and DevOps concepts.
I used this project to understand how cloud infrastructure, CI/CD pipelines,
and DevOps tools work together in a real project setup.

---------------------------------------------------------------------------
## Project Work Done

In this project, I practiced the following:
- Creating my own GitHub repository and pushing the project code
- - Practicing basic AWS services like EC2 and IAM
- Learning CI/CD pipeline concepts
- Understanding Docker container basics
- Learning Kubernetes deployment fundamentals
- Writing personal notes for revision and interview preparation
